原理图

Schematics 可以说是 Angular 生态中非常强大的一个功能。通过 Schematics 可以快速添加项目目录或者生成常用的业务模板。

目前 ng-matero 实现了 ng add 及 ng generate 功能，之后还会添加 ng update，方便 ng-matero 项目的升级维护，同时也会继续完善 ng generate，提供更多常用的业务模板。

  • ng add
  • ng generate
  • ng update
